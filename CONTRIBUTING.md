# Contributing to MorphoSNN Core

## Scope

MorphoSNN Core is a public seed repository for bio-inspired distributed neuromorphic control concepts, examples, benchmarks, and documentation.

## What Contributions Are Welcome

- documentation fixes;
- toy examples;
- benchmark protocol suggestions;
- references;
- issue reports.

## What Is Out of Scope

- confidential partner data;
- unpublished validation data;
- proprietary hardware files;
- claims of validated performance without evidence.

## Development Principles

- public-safe;
- reproducible;
- cautious claims;
- clear non-goals.

## Pull Request Checklist

- The contribution contains only public, non-confidential material.
- Any performance claim is supported by reproducible evidence.
- Any new reference is public and attributable.
- New examples can be run with documented commands.
- Non-goals and claim boundaries remain clear.
